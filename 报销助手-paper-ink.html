产品思考 · 为什么报销 Agent 值得做？

报销，是 AI Agent 的绝佳落地场景

理由一
需求边界收敛
有限集合，AI 极易做到 100% 精准覆盖

报销不是开放式任务。发票种类（滴滴、餐饮、机票）、需提取的字段（金额、日期、抬头）都是固定的有限集合，AI 极易做到 100% 精准覆盖。

理由二
规则白纸黑字
高度结构化，大模型最擅长执行清晰规则

像《2026 年度报销流程》都有明确 SOP 的文档，大模型最擅长的就是阅读理解并执行边界清晰的规则。只要规则明确，AI 就不会"胡编乱造"。

第 1 题 / 4 · 报销防坑测试

打车费的"薛定谔状态"

📋 情景题 小明常驻深圳。
周二在深圳拜访客户，打车花了 ¥50。
周四出差上海，从酒店打车去客户公司，花了 ¥30。

这两张发票，分别该选什么费用类型？
A 两张都选"市内交通费"
B 两张都选"差旅费 - 的士费"
C 深圳→"市内交通费"，上海→"差旅费 - 的士费"
D 深圳→"市内交通费"，上海→"差旅费 - 出差交通费"

✓ 答案 C：常驻城市打车 = 市内交通费；出差地打车 = 差旅费-的士费；往返机场/火车站才选"出差交通费"。选错直接被退！

第 2 题 / 4 · 报销防坑测试

去机场的"刺客"

📋 情景题 小红出差去北京：
① 从家打车到宝安机场 ¥100
② 从大兴机场打车去酒店 ¥120
③ 从酒店打车去客户公司 ¥50

三笔打车费，该怎么选类型？
A 只要出差，全选"差旅费 - 的士费"
B 去机场选"市内交通费"，后两笔选"差旅费 - 的士费"
C ①②选"差旅费 - 出差交通费"，③选"差旅费 - 的士费"

✓ 答案 C：往返机场、火车站 = 出差交通费；在出差地其他打车办事 = 的士费。是不是感觉像在考驾照科目一？

第 3 题 / 4 · 报销防坑测试

市内打车的"隐藏关卡"

📋 情景题 小强昨晚加班到 22:00，
打车回家花了 ¥60。

他报销"市内交通费"时，
只上传了一张滴滴电子发票并填了金额。

他的报销单会有什么下场？
A 顺利通过，坐等打款 🎉
B 被打回：必须补充"行程单"，并填写"事由"和"往返地"
C 被打回：加班打车不能报销

✓ 答案 B：市内交通费除发票外，必须附【行程单】，并手填"事由、上车点、下车点"。少一项退回重填——人的耐心就是这么耗尽的。

第 4 题 / 4 · 报销防坑测试

部门聚餐的"费用归属"

📋 情景题 小李作为组织者，带团队吃了一顿部门聚餐，共花了 ¥1,200。

填报销单时，他不确定分类，于是顺手选了"其他报销"下的"餐费"。

请问他的选择正确吗？
A 正确，有吃饭发票就选餐费，没问题
B 错误，应选"员工福利" → "部门团建"
C 错误，部门聚餐属私人消费，不能报销
D 错误，应选"行政费用" → "餐饮招待"

✓ 答案 B：部门聚餐属于员工福利范畴，正确路径是【员工福利 → 部门团建】。选"餐费"会触发费用类型不符被财务退回，"餐饮招待"则专指对外商务接待。
