第1题 / 3 · 报销防坑测试

打车费的"薛定谔状态"

📋 情景题 小明常驻深圳。
周二他在深圳拜访客户，打车花了 ¥50。
周四出差上海，从酒店打车去客户公司，花了 ¥30。

这两张滴滴发票，分别该选什么费用类型？
A 两张都选"市内交通费"
B 两张都选"差旅费 - 的士费"
C 深圳→"市内交通费"，上海→"差旅费 - 的士费"
D 深圳→"市内交通费"，上海→"差旅费 - 出差交通费"

✅ 答案 C：常驻城市打车 = 市内交通费；出差地打车 = 差旅费-的士费；往返机场/火车站才选"出差交通费"。选错直接被退！

第2题 / 3 · 报销防坑测试

去机场的"刺客"

📋 情景题 小红出差去北京：
① 从家打车到宝安机场 ¥100
② 从大兴机场打车去酒店 ¥120
③ 从酒店打车去客户公司 ¥50

这三笔打车费，该怎么选类型？
A 只要出差，全选"差旅费 - 的士费"
B 去机场选"市内交通费"，后两笔选"差旅费 - 的士费"
C ①②选"差旅费 - 出差交通费"，③选"差旅费 - 的士费"

✅ 答案 C：往返机场、火车站 = 【出差交通费】；在出差地其他打车办事 = 【的士费】。是不是感觉像在考驾照科目一？

第3题 / 3 · 报销防坑测试

市内打车的"隐藏关卡"

📋 情景题 小强昨晚加班到 22:00，
打车回家花了 ¥60。

他报销"市内交通费"时，
只上传了一张滴滴电子发票并填了金额。

请问他的报销单会有什么下场？
A 顺利通过，坐等打款 🎉
B 被打回：必须补充"行程单"，并填写"事由"和"往返地"
C 被打回：加班打车不能报销

✅ 答案 B：市内交通费除发票外，必须附【行程单】，并手填"事由、上车点、下车点"。少一项退回重填——人的耐心就是这么耗尽的。
